# Misaka

This project is based on the book "[ゼロからのOS自作入門](https://amzn.asia/d/0dcERgAx)".

## [temp] building the loader

```
$ cd $HOME/edk2
```

```
$ source edksetup.sh
```

```
$ build
```

## [temp] building the kernel

```
$ source $HOME/osbook/devenv/buildenv.sh
```

```
$ cd $HOME/workspace/misaka/kernel
```

```
$ make
```
